# Contact Book Application (C#)  
## Overview 
This project is a simple Contact Book application developed in C#.   
It allows users to manage their contacts efficiently with basic operations.  
## Features 
- **Add Contacts:** Save new contact details including name, phone number, email, and address.
- **Search Contacts:** Find contacts by name or other details.
- **Delete Contacts:** Remove unwanted contacts from the list.
- **Display Contacts:** Show all saved contacts in a clear list format.
- ## Technical Details
- Developed using **C# Windows Forms** for a simple graphical interface.
- Data is stored in memory for easy access and manipulation.
- Focused on usability and straightforward navigation.
